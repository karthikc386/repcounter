# RepCounter

A tiny web project for counting repetitions (reps).

## Overview
This project contains a single `index.html` that implements a simple rep counter UI.

## Usage
- Open `index.html` in a browser.
- Or serve the folder with a simple HTTP server and open `http://localhost:8000`.
  - Example (Python 3):

```powershell
python -m http.server 8000
```

## Project Files
- `index.html` — main UI and logic

## Development
Edit `index.html` to modify behavior or styling.
